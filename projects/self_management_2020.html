Some things I've tried (sucessfully)

Important principle: single source of truth

I always want a single source of truth because manually syncing between various media is an extra task that I'm not willing to do permanently. Throughout the years I've been going back and forth between propritizing either a calendar, a (kanban? not exactly.) board or lists of various sorts. Despite the advantages they all have, having single source of truth matters more, I'm sure I need a 'primary' medium.

Bullet Journal

One thing I want to point out is the Bullet Journal, which can be described as a todo-app as a book. By the time I was starting to use it (2016), I was already enjoying writing with fountain pens, which made the whole 'book instead of app' thing very inviting. It turned out immediately though that I had to adapt the protocol and use a computer calendar instead having the calendar in the book because I reschedule things too often and on a hand-written medium that is a mess and nit as prettily as I want my founten-pen-things to be like.
Important learning: use notebook with enumerated pages and an index. It helps writing things 'into the chaos' and still find them back. I find that handwritten media are particularly good when working on drafts of all sorts, for instance pseudo code and forst versions of technical drawings. This is great for me as a prototyper and therefore I use handwritten media (almost) daily.
